# Derivative Calculator
> Some scheme programs, I found them interesting and beautiful.

## Platform
* Base on `Mit-Scheme`.

## Introductions
* **`queens.scm`** : N Queens.
* **`deriv.scm`** : Calculate simple deriviate.
* **`Ackermann.scm`** : Ackermann function.
* **coin-count** : Count how many methods get a change.
